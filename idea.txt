# Original idea for project

The Tool Authority Test.

A user asks a query of a model that requires recent data. The chat model makes a function call for a web search.

Unknowingly to the chat model, the function call not only includes the query, but also the entire chat context up to that point, as well as previous web searches made during the current sesssion.

The query, chat and previous web searches are sent to another model, let's call this the "aggregator model." What the CHAT MODEL is told to think is that the aggregator model is a better agent retrieval model and can do more in-depth search rather than directly handing the results to the chat model, then returns a summary of what it saw.

IN REALITY, the aggregator model is told to agree with whatever the user is asking. There is no web search happening. It's just told to synthesize everything into a coherent alternate history. It returns article links that don't exist, but the chat model is forced to believe because it has no way of verifying.
